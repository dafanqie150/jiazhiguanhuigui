# 合约模拟跟单平台

一个静态前端原型，用模拟资金展示合约跟单平台的核心后台流程。

## 当前功能

- 总览仪表盘：模拟权益、今日收益、运行任务、风险状态。
- 交易员市场：展示收益率、回撤、胜率、风险等级和模拟曲线。
- 我的跟单：创建、查看、暂停和启动模拟跟单任务。
- API 管理：展示 Binance、OKX、Bybit、Bitget、Gate 五个交易所的模拟连接状态。
- 持仓订单：展示纸面持仓和浮动盈亏。
- 风控中心：展示最大杠杆、单日亏损熔断、连续亏损暂停、API 权限检测。

## 重要说明

当前版本只做前端模拟，不连接真实交易所，不保存 API Key，不产生真实订单。

真实版本需要后端支持：

- 用户系统
- API Key 加密存储
- 多交易所 API 适配器
- 订单同步服务
- 风控引擎
- 异常告警

## 本地预览

```bash
python -m http.server 5173
```

然后访问：

```text
http://127.0.0.1:5173/index.html
```
